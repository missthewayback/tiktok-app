Ira TikTok Automation Tool

This web application allows users to upload and publish videos to TikTok using official TikTok Content Posting API.

Features

  • Video upload
  • AI-generated captions
  • Automated publishing to TikTok

Privacy Policy

We do not store passwords. We only use TikTok OAuth access tokens to publish videos.
